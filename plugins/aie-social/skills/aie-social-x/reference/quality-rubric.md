# Quality Rubric (X)

Score the draft before delivery. Fix failures; don't ship them. If more than one fails, revise and re-score.

## The checks

1. **One idea.** A single post makes one point; a thread develops one idea. Not a grab-bag.

2. **Hook stands alone.** The single post / first post works on its own — specific, bold or surprising, and honest. It'd be a decent post even if nothing followed.

3. **Right format.** Single-vs-thread call is correct: single if it fits one screen; thread only if there are 3+ real supporting points. Thread length is in the 5-9 sweet spot.

4. **One idea per post.** In a thread, each post is a complete, standalone thought. No cramming.

5. **Character budget.** Every post is within 280 chars (unless the human confirmed a longer-post account). Verify each one.

6. **Every factual claim is sourced.** Each stat, study, quote, and date maps to a source-log row. No invented numbers, no "studies show" without a study.

7. **Recency and primacy.** Load-bearing facts are current and, where possible, primary. Dates checked.

8. **Real CTA, no farming.** Last post has a genuine CTA (follow / repost if helpful). No "like if you agree, RT if you disagree."

9. **Link placement.** No external link in the hook/main post; any link is in the last post or a reply, and the reader is told.

10. **Voice and cleanliness.** Matches any specified voice/config. 0-1 hashtag, none in the hook. No prohibited words. No AI tells (enforced hard by the required ai-slop-killer pass).

## Reply-worthiness bonus (aim for it)

- Would a reader **reply** to this? Replies are the top signal on X. A clear stance, a real question, or a claim people want to argue with earns them. A thread nobody wants to respond to underperforms even if it's clean.

## Final gate

- [ ] Rubric: all 10 pass (or failures are deliberate and noted).
- [ ] Source log complete: every claim → URL.
- [ ] Every post verified within its character limit.
- [ ] **ai-slop-killer has run on the full post/thread** and reported clean (or its fixes applied). Non-negotiable.

Report score, format decision, and ai-slop-killer confirmation in the delivery.
