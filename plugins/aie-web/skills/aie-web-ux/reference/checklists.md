# UX Review Checklists

## Navigation audit

- [ ] Every page reachable in 3 clicks or fewer from the homepage.
- [ ] Main navigation consistent across all pages.
- [ ] Mobile menu easy to open, close, and navigate.
- [ ] Breadcrumbs on multi-level pages.
- [ ] Current page clearly highlighted in nav.
- [ ] Logo/home link on every page.
- [ ] Nav items grouped logically, not random.
- [ ] Dropdowns only where genuinely needed.
- [ ] Footer includes sitemap or key links.
- [ ] Search (if present) is discoverable and works.

## Visual hierarchy

- [ ] Eye flows to important content (F- or Z-pattern).
- [ ] Primary CTA prominent and above the fold.
- [ ] Clear visual separation between sections.
- [ ] H1/H2/H3 form a real hierarchy, sized for meaning not decoration.
- [ ] Whitespace gives breathing room.
- [ ] Key information not buried in walls of text.
- [ ] Color used purposefully, not just decoration.
- [ ] Images support content rather than replace it.

## Mobile UX

- [ ] Touch targets at least 44×44 px.
- [ ] Key actions reachable in the thumb zone.
- [ ] No hover-dependent interactions (mobile has no hover).
- [ ] Forms use correct input types (email, tel, number) for the right keyboard.
- [ ] Images scale responsively; no horizontal scroll.
- [ ] Body text at least 16px and readable.
- [ ] Tap-to-call and tap-to-email work.
- [ ] Mobile menu opens without layout shift.

## Readability

- [ ] Line length ~45–75 characters.
- [ ] Body text at least 16px.
- [ ] Line spacing generous (1.5–1.75).
- [ ] Paragraphs short (3–5 sentences).
- [ ] Subheadings roughly every 200–300 words.
- [ ] Contrast meets WCAG AA (4.5:1 for normal text).
- [ ] Readable font (sans-serif usually best for body on web).
- [ ] Ragged-right, not justified.
- [ ] Lists used to break up information.
- [ ] Bold/italic used sparingly and purposefully.

## Anti-patterns to flag

- Walls of text — long paragraphs with no headings or lists.
- Mystery-meat navigation — icons with no labels.
- Inconsistent button styles — no clear clickable standard.
- Auto-playing media — video/audio starting without consent.
- Popup overload — multiple overlays competing at once.
- Purposeless infinite scroll — no way to reach the footer.
- Hidden contact info — buried in one page or the footer only.
- Hover-dependent content — breaks on touch devices.
- No mobile menu — desktop nav unusable on phones.
- Slow first impression — 3+ seconds to feel interactive.
- Orphaned pages — content with no navigation path in.
- "Click here" links — anchor text that doesn't describe the destination.
